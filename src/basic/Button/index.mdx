# Button

Button 组件

## 类型

## 尺寸

3种尺寸，S、M、L 

S：小尺寸，高度为28px，宽度自适应，min-width为48px，padding为8px 

M：正常按钮，高度为32px，宽度自适应，min-width为64px，padding为16px 

L：大尺寸，高度为40px，宽度自适应，min-width为80px，padding为16px，一般用于全局操作
