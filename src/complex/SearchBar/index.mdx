# SearchBar

内部使用 `Form`。

## 基本用法

传入需要的搜索条件参数，生成对应的 SearchBar。

<Playground>
  <SearchBar
    onSearch={(values) => {
      console.log(values)
    }}
    fields={[{
      type: 'input',
      key: 'name',
      title: '名称',
      rules: [{
        required: true
      }],
      onChange: (form) => form.resetFields(['sex'])
    }, {
      type: 'select',
      key: 'sex',
      title: '性别',
      items: [{ value: 1, mean: '男' }, { value: 2, mean: '女' }]
    }]} 
  />
</Playground>
