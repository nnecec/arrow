# SearchBar

内部使用`Form`构建的`SearchBar`。

## 基本用法

传入需要的搜索条件参数，生成对应的 SearchBar。

<Playground>
  <SearchBar
    onSearch={(values) => {
      console.log(values)
    }}
    fields={[{
      type: 'input',
      key: 'name',
      title: '姓名',
    }, {
      type: 'select',
      key: 'type',
      title: '国籍',
      items: [{ value: 1, mean: '中华人民共和国' }, { value: 2, mean: '美利坚合众国' }],
      style: {
        width: 200
      }
    }]} 
  />
</Playground>

## 输入项 校验/联动

输入`名称`重置`性别`，校验`名称`输入是否符合规则。

`rules`的规范与`Form`中的`rules`一致。

<Playground>
  <SearchBar
    onSearch={(values) => {
      console.log(values)
    }}
    fields={[{
      type: 'input',
      key: 'name',
      title: '姓名',
      rules: [{
        required: true, message: '姓名必填',
      }, {
        max: 4, message: '姓名最大长度为4个字符'
      }],
      onChange: (form) => form.resetFields(['sex'])
    }, {
      type: 'select',
      key: 'sex',
      title: '性别',
      initialValue: 1,
      items: [{ value: 1, mean: '男' }, { value: 2, mean: '女' }]
    }]} 
  />
</Playground>
